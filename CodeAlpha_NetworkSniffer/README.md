# Network Sniffer Tool (CodeAlpha Task 1)

## 📌 Description
A Python-based network sniffer that captures and analyzes network traffic in real-time using Scapy.

## ⚙ Features
- Capture live packets
- Display Source & Destination IP
- Detect protocols (TCP/UDP)
- Payload preview
- GUI Dashboard
- Search & Sorting

## 🛠 Technologies Used
- Python
- Scapy
- Tkinter

## ▶ How to Run
```bash
pip install scapy
python sniffer.py
